# Evaluación Parcial 2 - Administración de Sistemas Operativos
**Integrantes:** Gerardo Grados - Benjamin Arriagada

## Resumen del Proyecto
Este repositorio contiene los archivos de configuración para un entorno de red basado en Windows Server 2019.

### Servicios Implementados:
1. **Active Directory:** Dominio `Grados.lab` con UOs jerárquicas y restricciones de acceso.
2. **IIS:** Servidor web con página institucional personalizada.
3. **FTP:** Servidor de archivos con acceso autenticado para integrantes.
4. **Conectividad:** Pruebas exitosas desde cliente Windows 7.
